# U-Devs-E-Commerce-
This project is a front-end E-Commerce Web Application designed to simulate an online shopping experience. It allows users to browse products, view product details, manage their shopping cart, and proceed through a checkout flow. The application focuses on a clean UI, responsive design, and user-friendly navigation using modern web technologies.

The project is ideal for learning and demonstrating core concepts of web development such as page navigation, UI components, layout design, and basic e-commerce flow.
The project is built using the following technologies:

HTML5 – Structure of web pages

CSS3 – Styling and layout customization

Bootstrap 5 – Responsive design and UI components

Bootstrap Icons – Icons for cart, ratings, and UI elements

JavaScript (Basic / Bootstrap JS) – Interactivity and component behavior
